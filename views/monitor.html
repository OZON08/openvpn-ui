client-disconnect webhook

          

            
OpenVPN calls this script on every disconnect, posting final byte counts to /api/v1/monitor/disconnect (authenticated via X-Monitor-Token). Pre-built script: build/assets/client-disconnect.sh. Wire it into server.conf with:


script-security 2
client-disconnect /etc/openvpn/scripts/client-disconnect.sh

            
Then set OPENVPN_UI_HOOK_TOKEN on the openvpn container and OPENVPN_UI_MONITORING_HOOK_TOKEN (same value) on the openvpn-ui container.
